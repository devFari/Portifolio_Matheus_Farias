Data Science - Modelagem de dados com Python

Python para Data Science: Primeiros passos

Neste curso, você vai entender as funções das bibliotecas e pacotes na linguagem Python, explorando como utilizá-los para facilitar e organizar seu código. Será apresentado o uso das built-in functions e suas diversas utilidades, além de como criar funções personalizadas para tornar seus programas mais modulares e reutilizáveis. Você também aprenderá a trabalhar com estruturas de dados compostas e aninhadas, construindo listas e dicionários de forma eficiente com list e dict comprehension. Por fim, o curso aborda os diferentes tipos de erros e exceções em Python, ensinando a identificar e tratar comportamentos indesejados para garantir um código mais robusto e funcional.
