Inteligência de Negócios (BI) (FIAP)

Neste curso de MySQL, você vai aprender a dominar uma das ferramentas de gerenciamento de banco de dados mais utilizadas no mercado, passando desde a instalação e o acesso por linha de comando até o uso de sua interface gráfica. Ao longo do curso, será possível criar bancos de dados e tabelas, desenvolver consultas SQL para extrair insights valiosos e aplicar comandos para realizar modificações estruturais. Essa formação oferece a base essencial para quem deseja compreender e utilizar o MySQL de forma prática e eficiente no dia a dia profissional.
