Estatística com Python: resumindo e analisando dados

Neste curso, você aprenderá a aplicar os conceitos básicos de estatística para gerar insights e resolver problemas de forma eficaz, compreendendo a importância de avaliar cada tipo de dado de maneira adequada para extrair informações relevantes. Serão apresentadas as principais medidas de tendência central, separatrizes e dispersão para realizar uma estatística descritiva completa, além de explorar a análise da distribuição dos dados e o que seus comportamentos podem indicar. Você também conhecerá o potencial da linguagem Python como ferramenta de suporte para a análise estatística, contribuindo para processos de tomada de decisão baseados em dados.

Certificado

Estatística com Python: probabilidade e amostragem

Neste curso, você aprenderá conceitos fundamentais de probabilidade e amostragem aplicados à análise de dados com Python. Será apresentado o funcionamento das distribuições binomial, Poisson e normal, além do entendimento sobre nível e intervalo de confiança para interpretar resultados de forma mais precisa. Você conhecerá diferentes técnicas de amostragem e aprenderá a calcular o tamanho ideal da amostra para garantir representatividade e confiabilidade nas análises, utilizando a linguagem Python como ferramenta prática para aplicação desses conceitos.

Certificado

Data Science: testando relações com Regressão Linear

Neste curso, você aprenderá a identificar relações lineares entre variáveis e a diferenciar variáveis explicativas de variáveis resposta. Será apresentado o processo de ajuste de um modelo de regressão linear, a interpretação de seus coeficientes e do coeficiente de determinação R², além de como utilizar o modelo para realizar previsões. Você também aprenderá a analisar a qualidade do modelo, avaliando sua capacidade de representar os dados e gerar resultados confiáveis, utilizando ferramentas práticas de Data Science para aplicação desses conceitos.

Certificado

Regressão Linear: técnicas avançadas de modelagem

Neste curso, você aprenderá técnicas avançadas de modelagem em regressão linear, aplicando transformações nos dados antes do treinamento para melhorar o desempenho dos modelos. Será apresentado o uso de bibliotecas como Statsmodels e Scikit-learn para realizar regressões, obter previsões pontuais e interpretar corretamente os coeficientes estimados. Além disso, você desenvolverá análises gráficas para avaliar e interpretar os resultados, aprimorando sua capacidade de criar modelos mais precisos e adequados às características dos dados.

Certificado

Classificação: aprendendo a classificar dados com Machine Learning

Neste curso, você aprenderá a realizar análises exploratórias de dados voltadas para projetos de machine learning, aplicando transformações em variáveis categóricas para preparar as informações para os modelos. Será apresentado o processo de treinamento de modelos de classificação a partir de dados reais, bem como a utilização desses modelos para prever novos casos. Você também aprenderá a armazenar tanto modelos de transformação de dados quanto modelos de machine learning em arquivos pickle, garantindo a reutilização e a praticidade na implementação das soluções.

Certificado

Classificação: validação de modelos e métricas de avaliação

Neste curso, você aprenderá a validar modelos de machine learning utilizando o método hold out e a validação cruzada, garantindo uma avaliação mais precisa do desempenho. Serão apresentadas diferentes métricas para avaliar modelos, com orientação sobre quais são mais adequadas para cada tipo de problema. Você também conhecerá técnicas de balanceamento de dados, como oversampling e undersampling, e aprenderá a aplicar pipelines para validar modelos de forma correta e organizada, assegurando maior confiabilidade nos resultados obtidos.

Certificado

IA aumentada: prevendo atrasos de voos

Neste curso, você aprenderá a aplicar técnicas de visualização de dados para comunicar insights de forma clara e objetiva, além de desenvolver habilidades na aplicação de métodos de codificação para transformar variáveis categóricas. Será apresentado como implementar modelos de regressão, incluindo o DummyRegressor e o RandomForestRegressor, e como analisar seus resultados por meio de métricas e gráficos. Você também aprenderá a identificar e selecionar as variáveis mais importantes para o modelo de machine learning e a explorar a otimização de hiperparâmetros para aprimorar o desempenho, utilizando essas técnicas para prever atrasos de voos de maneira mais precisa e eficiente.

Challenge Telecom X: análise de evasão de clientes - Parte 2

Neste desafio Telecom X – Parte 2, você assumirá o papel de cientista de dados, avançando da análise exploratória realizada na primeira etapa para a construção de modelos preditivos capazes de prever a evasão de clientes (churn). A partir do dataset previamente tratado, você irá preparar os dados, selecionar as variáveis mais relevantes, criar e avaliar modelos demachine learning utilizando bibliotecas como Python, Pandas, Scikit-learn e ferramentas de visualização. O objetivo é identificar quais clientes têm maior risco de evasão, quais fatores influenciam esse comportamento e quais perfis merecem maior atenção por parte da empresa. Ao longo do desafio, você contará com o Trello para organizar as tarefas e seguirá um fluxo estruturado, desde a criação de um novo repositório no GitHub até a elaboração de um relatório final completo, destacando os resultados obtidos e as recomendações estratégicas para reduzir o churn. Este projeto consolida todos os conhecimentos adquiridos na especialização em Data Science, colocando em prática técnicas de ETL, análise de dados, modelagem preditiva e interpretação de resultados para tomada de decisão baseada em dados.
