Data Science - Modelagem de dados com Python

Python para Data Science: Primeiros passos

Neste curso, você vai entender as funções das bibliotecas e pacotes na linguagem Python, explorando como utilizá-los para facilitar e organizar seu código. Será apresentado o uso das built-in functions e suas diversas utilidades, além de como criar funções personalizadas para tornar seus programas mais modulares e reutilizáveis. Você também aprenderá a trabalhar com estruturas de dados compostas e aninhadas, construindo listas e dicionários de forma eficiente com list e dict comprehension. Por fim, o curso aborda os diferentes tipos de erros e exceções em Python, ensinando a identificar e tratar comportamentos indesejados para garantir um código mais robusto e funcional.

Praticando Python para Data Science: Challenge Alura Store

O desafio Allure Store, análise de vendas e desempenho de lojas, tem como objetivo ajudar o Senhor João a decidir qual loja vender para investir em um novo negócio. Sabemos que grandes empresas, e-commerces e até mesmo você, mesmo sem perceber, tomam decisões baseadas em dados. E é aí que vem o nosso objetivo. Você foi contratado como pessoa analista de dados para ajudar Sr. João a decidir qual loja ele deve vender. Para isso, você vai analisar as seguintes métricas: faturamento total de cada loja, categorias mais populares, média de avaliação dos clientes, produtos mais e menos vendidos e custo médio do frete. Com essas métricas, você será capaz de decidir qual loja Sr.João deve vender.

Modelagem de dados: desenvolvendo o modelo lógico

Neste curso, você vai aprender a construir um modelo físico na modelagem de dados, entendendo como escolher os tipos de dados adequados, definir limites de caracteres e representar corretamente os relacionamentos e suas cardinalidades. Modelagem de Dados com Python G8 - ONE

A formação Modelagem de Dados com Python foi criada para desenvolver sua habilidade em analisar, estruturar e modelar dados com precisão, utilizando a linguagem Python como aliada no processo. Você começará pelos fundamentos de Python voltados à ciência de dados, explorando estruturas, funções e tratamento de exceções. Em seguida, colocará seus conhecimentos em prática com um desafio real no projeto Alura Store, assumindo o papel de analista de dados. A jornada continua com o domínio da modelagem de dados: desde a identificação de entidades, atributos e relacionamentos, passando pela criação do modelo lógico com aplicação de padrões e relacionamentos por chaves, até a construção do modelo físico, considerando tipos de dados, restrições e integração com ferramentas. Ao final, você estará preparado para aplicar boas práticas de modelagem em projetos reais, conectando a análise de dados à estrutura eficiente de bancos.
