Data Science - Aprendendo a fazer ETL

NumPy: análise numérica eficiente com Python

Domine os fundamentos da biblioteca Numpy e potencialize suas análises com Python! Neste curso, você vai aprender a ler e escrever dados usando Numpy, criar e manipular arrays multidimensionais, gerar números aleatórios e controlar a aleatoriedade com seeds. Além disso, vai explorar funções de agregação e realizar cálculos de regressão linear, consolidando conhecimentos essenciais para trabalhar com dados de forma eficiente e estruturada.

Certificado

Pandas: conhecendo a biblioteca

Domine os recursos essenciais da biblioteca Pandas e aprenda a desenvolver um projeto completo de análise de dados! Neste curso, você vai realizar análises exploratórias, construir diferentes tipos de gráficos e aprender a selecionar dados específicos para obter insights relevantes. Também vai saber como lidar com dados nulos, remover linhas e colunas desnecessárias e criar colunas personalizadas, tudo isso utilizando ferramentas práticas e eficientes do Pandas para potencializar seus projetos em ciência de dados.

Certificado

Pandas I/O: trabalhando com diferentes formatos de arquivos

Aprenda a lidar com diferentes formatos de dados e aprofunde suas habilidades em manipulação de dados com Python! Neste curso, você vai entender o que são arquivos CSV, XLSX, JSON, HTML e XML, e como trabalhar com esses formatos tanto na entrada quanto na saída de dados. Também aprenderá a ler informações diretamente do Google Planilhas, extrair tabelas de páginas web, criar e escrever em um banco de dados local e realizar consultas SQL, dominando etapas fundamentais para projetos de análise e ciência de dados.

Certificado

Pandas: transformação e manipulação de dados

Aprofunde seus conhecimentos em transformação de dados com Pandas e aprenda a manipular informações de forma eficiente para construir projetos sólidos em ciência de dados. Neste curso, você vai entender e aplicar comandos essenciais de transformação, diferenciar os tipos de informação nos dados, manipular valores numéricos e utilizar métodos específicos para lidar com dados textuais. Também explorará transformações com dados do tipo *datetime* e entenderá como as conversões de dados impactam diretamente na qualidade e nos resultados dos seus projetos analíticos.

Certificado

Data Visualization: criando gráficos com bibliotecas Python

Este curso é a oportunidade perfeita para quem deseja criar gráficos impactantes e profissionais para visualização de dados. Nele, você vai aprender a utilizar a biblioteca Matplotlib, entendendo o funcionamento de figuras e eixos e personalizando suas criações com cores, estilos e anotações. Além disso, vai explorar o potencial do Seaborn para gráficos mais sofisticados e interativos com a biblioteca Plotly. Também vai descobrir as melhores práticas para transformar dados em insights visuais poderosos, garantindo apresentações claras e impactantes para seus projetos e análises.

Certificado

Challenge Telecom X: análise de evasão de clientes

Neste desafio Telecom X, você assumirá o papel de assistente de análise de dados em um projeto real de Data Science, com foco na identificação das causas da evasão de clientes (churn) em uma empresa de telecomunicações. Você será responsável por executar todo o processo de ETL: desde a extração dos dados diretamente de uma API, até a transformação e limpeza das informações para realizar uma análise exploratória detalhada. Durante o curso, você colocará em prática o uso de bibliotecas como Pandas, Seaborn e Matplotlib, criando visualizações e insights que ajudarão a equipe de Data Science a desenvolver modelos preditivos para reduzir a perda de clientes. Ao final, você organizará suas descobertas em um relatório completo, detalhando os problemas identificados, os padrões encontrados e sugestões de ações para melhorar os resultados da empresa.

Certificado Repositorio Challenger

IA aplicada a Data Science: utilizando IA na construção de visualização de dados

Neste curso, você aprenderá a utilizar o ChatGPT como um assistente pessoal na análise e visualização de dados, otimizando tempo e ampliando suas possibilidades criativas. Descubra como construir visualizações de dados personalizadas para diferentes públicos, entendendo as limitações e as potencialidades das IAs na criação de gráficos e visuais diversos. Além disso, você vai explorar ferramentas como o Gamma e o próprio ChatGPT para estruturar e apresentar relatórios completos e impactantes a partir dos dados analisados.

Certificado

Aprendendo a fazer ETL G8 - ONE

Neste segundo treinamento, você aprenderá o método ETL (Extração, Transformação e Carregamento), essencial para trabalhar com dados de forma eficiente e estruturada. Exploraremos como extrair informações de diferentes fontes, realizar o processamento necessário para limpar, organizar e transformar os dados, e, por fim, carregá-los em formatos adequados para análise. Esse processo é fundamental para garantir a qualidade dos dados e gerar insights que apoiem a tomada de decisões estratégicas. Este curso faz parte do Programa ONE, uma parceria entre a Alura Latam e a Oracle.

Certificado
