Dashboard Financeiro - Controle Simplificado de Finanças

Muitas pessoas e pequenos negócios enfrentam dificuldades para organizar e visualizar suas finanças de forma clara e acessível, perdendo o controle sobre gastos, receitas e metas financeiras. Esta solução oferece um sistema intuitivo de registro de transações através de um formulário dinâmico, que armazena os dados localmente em arquivo JSON, permitindo a geração de um dashboard interativo com gráficos e métricas essenciais.

Desenvolvido com HTML, CSS e JavaScript, o projeto prioriza a simplicidade e adaptabilidade, funcionando diretamente no navegador sem dependências externas. A arquitetura modular permite que usuários personalizem categorias e relatórios conforme suas necessidades, seja para controle doméstico ou gestão de pequenos empreendimentos. A visualização dos dados em tempo real facilita a identificação de padrões de gastos, otimizando o planejamento financeiro de forma prática e acessível.

Serviço de Streaming de Exames em Tempo Real

A necessidade de acompanhamento remoto e em tempo real de exames de ressonância magnética por médicos especialistas motivou o desenvolvimento desta solução inovadora para a Clínica Lucilo Ávila. O sistema permite que médicos visualizem exames diretamente de qualquer local com acesso à VPN da clínica, garantindo agilidade na avaliação, orientação durante procedimentos e maior qualidade no diagnóstico.

Utilizando Node.js no backend e uma interface web responsiva com HTML5, CSS3 e JavaScript, a plataforma recebe e transmite fluxos de vídeo em tempo real a partir de codificadores J-Tech (HDMI para IP), que convertem o sinal dos aparelhos de RM para formato H.264, optimizado para baixo consumo de banda. A solução é segura, restrita à rede VPN da clínica, e não armazena dados, assegurando conformidade com requisitos de privacidade.

Desenvolvido para operar em múltiplas unidades, o serviço integra tecnologia acessível e eficiente, substituindo modelos tradicionais de acompanhamento presencial e elevando o padrão de atendimento e suporte em ambientes médicos especializados.

Seletor de Cardápio Personalizado - React App

A dificuldade em criar planos alimentares que considerem objetivos nutricionais específicos, restrições alimentares e preferências individuais inspirou o desenvolvimento desta aplicação React. O Seletor de Cardápio Personalizado utiliza um formulário inteligente para capturar dados biométricos, metas (como perda de peso ou ganho de massa) e preferências do usuário, cruzando essas informações com um banco de dados nutricional em JSON para gerar cardápios sob medida.

Desenvolvido com React 18+, JavaScript e CSS3, o sistema emprega hooks como useState e useEffect para gerenciar o estado da aplicação e filtrar opções alimentares com base nas necessidades do usuário. A interface responsiva e intuitiva exibe refeições organizadas, detalhando informações nutricionais para cada escolha. Ideal para quem busca alimentação balanceada sem complicação, a solução demonstra eficiência no uso de tecnologias modernas para entregar valor real com simplicidade e personalização.

Healer Saúde Integrada - Sistema de Gestão

A gestão manual de agendamentos, cadastro de pacientes e controle de informações em clínicas médicas representa um desafio significativo para a eficiência operacional. O Sistema de Gestão Healer Saúde Integrada foi desenvolvido para solucionar essa dor, oferecendo uma plataforma completa e intuitiva que digitaliza processos críticos de clínicas e consultórios.

Com funcionalidades como cadastro de pacientes, agendamento inteligente de exames, controle de histórico e pesquisa avançada com filtros, o sistema melhora a organização, reduz erros e economiza tempo. Desenvolvido em Node.js, Express, JavaScript, HTML5 e CSS3, e utilizando JSON como banco de dados, a solução é leve, de fácil implementação e personalizável, atendendo desde pequenos consultórios até clínicas de médio porte. A interface responsiva e segura, com autenticação de usuários, garante acesso controlado e confiabilidade no manejo de informações sensíveis. Ideal para profissionais de saúde que buscam eficiência e modernização em sua gestão diária.

Healer PET - Sistema de Gestão para Animais de Estimação

O Healer PET é uma solução completa desenvolvida para simplificar e organizar o cuidado com animais de estimação. A aplicação web responde à necessidade de muitos tutores que enfrentam dificuldades para gerenciar e acompanhar as diversas atividades necessárias para o bem-estar de seus pets, desde cuidados básicos até acompanhamentos veterinários.

Com uma interface intuitiva e amigável, o sistema permite o registro de 16 tipos de atividades diferentes, incluindo passeios, alimentação, medicamentos, banho, veterinário e viagens. Cada ação é armazenada em um histórico organizado, proporcionando uma visão clara e temporal de todos os cuidados prestados ao animal. Desenvolvido com Node.js, Express, JavaScript, HTML5 e CSS3, e utilizando JSON para armazenamento de dados, o Healer PET é leve, responsivo e funciona de modo offline, garantindo acesso rápido e seguro sem dependência de internet.

A solução é ideal para tutores que desejam manter um controle detalhado e personalizado da rotina de seus pets, assegurando que nenhum cuidado seja negligenciado e melhorando a qualidade de vida dos animais através de uma gestão eficiente e afetuosa.
